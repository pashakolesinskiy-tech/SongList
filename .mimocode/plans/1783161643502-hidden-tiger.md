# Оптимизации и улучшения Druisk

## 1. Устранение дублирования кода

### 1.1 Вынести `esc()` в общий модуль
Функция `esc()` определена в 3 файлах: `home.js:257`, `song.js:146`, `edit.js:161`.

**Решение:** создать `js/utils/escape.js`:
```js
export function esc(text) {
  const d = document.createElement('div');
  d.textContent = text || '';
  return d.innerHTML;
}
```

**Файлы:** `home.js`, `song.js`, `edit.js` — заменить локальные `esc()` на импорт.

### 1.2 Вынести обработчик удаления
Обработчик `.js-delete` дублируется в `renderList` и `renderSearchResults` (`home.js`).

**Решение:** вынести в отдельную функцию:

```js
function bindDeleteButtons(container, onDeleted) {
  container.querySelectorAll('.js-delete').forEach(btn => {
    btn.addEventListener('mousedown', (e) => e.preventDefault());
    btn.addEventListener('click', (e) => { ... });
  });
}
```

## 2. Производительность

### 2.1 Debounce поиска
Сейчас поиск срабатывает на каждое нажатие клавиши. При 100+ песнях это тормозит.

**Решение:** добавить debounce 200ms в `home.js`:
```js
let searchTimeout;
searchInput.addEventListener('input', () => {
  clearTimeout(searchTimeout);
  searchTimeout = setTimeout(() => { ... }, 200);
});
```

### 2.2 Ленивая загрузка песен
При большом списке — рендерить первые 50, остальные при скролле.

**Решение:** добавить `IntersectionObserver` или простой `scrollTop` детект в `home.js`.

## 3. UX улучшения

### 3.1 Счётчик песен в списке
Показать "12 песен" или "3 из 12" при поиске.

**Файл:** `home.js` — добавить `<div class="song-count">` после заголовка.

### 3.2 Индикатор загрузки при сохранении
Сейчас при сохранении кнопка не блокируется, можно нажать дважды.

**Файл:** `upload.js`, `edit.js` — добавить `disabled` и текст "Сохранение..." на время запроса.

### 3.3 Подтверждение при выходе из редактора
Если пользователь изменил текст и нажал "Назад", потеряются изменения.

**Файл:** `edit.js` — добавить `beforeunload` и проверку при клике на "Назад".

### 3.4 Кнопка "Наверх"
При длинной песне — кнопка возврата вверх.

**Файл:** `song.js` — добавить `scroll-to-top` кнопку.
