# Иконка сердечка: размер + фокус

## Проблемы
1. Сердечко слишком узкое — `♥` при `font-size: 1.1rem` мелкое
2. При нажатии на сердечко фокус уходит на другой элемент (родительский `<a>`) — кнопка внутри ссылки, браузер ставит фокус на `<a>`

## Решение

### 1. `css/style.css` — увеличить сердечко
`.btn-fav` (строка 167):
- `font-size: 1.1rem` → `font-size: 1.4rem`
- Добавить `line-height: 1` для точного размера

### 2. `js/views/home.js` — убрать фокус
Обработчик `js-fav` (строка ~262):
- `mousedown` обработчик уже есть с `preventDefault()` — это блокирует фокус по `mousedown`
- Добавить в `click` обработчик `e.currentTarget.blur()` после toggle — снимает фокус если он уже встал

Также в `renderList` — добавить `tabindex="-1"` на кнопку сердечка, чтобы она не была в tab-порядке и не получала фокус при клике.

## Файлы
- `css/style.css` — размер сердечка
- `js/views/home.js` — blur + tabindex

## Верификация
1. Сердечко заметно шире чем было
2. Клик по сердечку — сердечко меняет цвет, фокус НЕ уходит на строку песни
3. Клик по самой строке песни — переход к песне работает как раньше
