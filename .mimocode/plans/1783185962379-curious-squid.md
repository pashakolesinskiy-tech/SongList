# Исправление hover на iOS Safari

## Проблема
На iOS Safari `:hover` "застревает" после тапа — подсветка не сбрасывается пока не тапнуть по другому элементу.

## Решение
`css/style.css` — обернуть hover в `@media (hover: hover)`, чтобы он работал ТОЛЬКО на устройствах с мышью:

```css
/* БЫЛО */
.btn-fav:not(.is-fav):hover {
  color: #ef4444;
  transform: scale(1.15);
}

/* СТАЛО */
@media (hover: hover) {
  .btn-fav:not(.is-fav):hover {
    color: #ef4444;
    transform: scale(1.15);
  }
}
```

На тач-устройствах hover отключается — застревание исчезает. `:active` (scale при нажатии) продолжает работать на всех устройствах.

## Верификация
1. Desktop: hover на пустом сердечке — подсвечивает красным
2. iOS: тап по пустому сердечку — кратковременная подсветка (:active), hover не застревает
3. Тап по заполненному — сразу серое, без залипания
